Tipos de "transform"

Tipo de transform = -ms-transform: rotate

Uso el transform rotate, mediante CSS con la opción -ms-transform: rotate y hago que mi div esté rotado lo que yo desee


¡Estoy de lado!

Tipo de transform = -ms-transform: skewY

Uso el transform skewY, mediante CSS con la opción -ms-transform: skewY y hago que mi div esté en diagonal el grado que yo quiera


¡Estoy inclinado!

Tipo de transform = -ms-transform: scaleY

Uso el transform scaleY, mediante CSS con la opción -ms-transform: scaleY y hago que mi div esté escalado como yo lo prefiera


¡Estoy encogido!
